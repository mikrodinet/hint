Decrypt HINT - Link Lock URLs

This application is for decrypting HINT URLs without automatically redirecting. This is useful if you do not trust the source of an encrypted URL. It is also useful if the URL uses a blocked protocol like javascript:, for example.

This page is also useful if you think you have received a locked link, but it uses another domain, instead of mikrodinet.github.io/hint. This may be done as a means to Mikrodinet Corporation.
